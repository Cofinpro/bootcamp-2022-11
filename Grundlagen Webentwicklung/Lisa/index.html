Taxonomy

Let's make a list with a colored reversed roman counter.
  1. Species: Felis catus
  2. Genus: Felis
  3. Subfamily: Felinae
  4. Family: Felidae
  5. Suborder: Feliformia
  6. Order: Carnivora
  7. Class: Mammalia
Here we have the problem, that we need to know, how many items will exist and place this magic number in CSS. Another way to create such a list would be:
  1. Species: Felis catus
  2. Genus: Felis
  3. Subfamily: Felinae
  4. Family: Felidae
  5. Suborder: Feliformia
  6. Order: Carnivora
  7. Class: Mammalia
Since <li> contains the counter/bullet and the text itself, we have to place the text in <span>s. But I think it is justifiable that we need to split the counter and text structurally and therefore edit the html-structure.
